Restaurant Food Ordering & Delivery System

A modern, full-stack Laravel web application for home-based food businesses to manage orders, inventory, and deliveries. Built with simplicity and scalability in mind.

## ✨ Features

### 👥 Customer Features
- **Browse Menu**: Browse products organized by categories
- **Shopping Cart**: Add/remove items, adjust quantities with session-based cart
- **Checkout**: Simple guest checkout (no account required)
- **WhatsApp Integration**: Direct order placement via WhatsApp with pre-filled order details
- **Order Tracking**: View order status and history
- **Real-time Updates**: Order status notifications

### 🛠️ Admin Features
- **Product Management**: Create, edit, delete products with images
- **Category Management**: Organize products into categories
- **Order Management**: View, filter, and update order statuses
- **Inventory Tracking**: Monitor stock levels and low-stock alerts
- **Sales Reports**: Track daily/monthly sales and top-selling products
- **Dashboard**: Real-time sales metrics and pending orders

### 🚀 Technical Highlights
- **Session-Based Cart**: Fast, lightweight cart system (no database overhead)
- **Service Layer Architecture**: Clean separation of business logic
- **Database Transactions**: ACID compliance for order creation
- **WhatsApp Click-to-Chat**: Free WhatsApp integration (no API keys needed)
- **Price Snapshots**: Historical order accuracy (prevents price manipulation)
- **Responsive Design**: Mobile-friendly Bootstrap + Tailwind CSS
